# business-intelligence-msc

Business Intelligence coursework from my Data Science MSc at 
Dalarna University, covering data warehousing, ETL, data mining, 
predictive modelling, text analytics, and PowerBI dashboards.

## About
This repository contains all lab work, assignments, and project 
deliverables completed as part of the Business Intelligence 
(AMI295) course at Dalarna University. The course covers the 
concepts and methods behind implementing business intelligence 
systems for data-driven decision making.

## Repository Structure
| Folder | Description |
|--------|-------------|
| `lab1-python-data-handling` | Introduction to Python and data handling |
| `lab2-powerbi-sales-dashboard` | Sales dashboard built in PowerBI |
| `lab3-classification` | Classification problems in BI |
| `lab4-clustering` | Clustering problems in BI |
| `lab5-text-mining-nlp` | Text mining and Natural Language Processing |
| `case-study-project` | Group research project and case study |

## Topics Covered
- Business Intelligence concepts and decision support systems
- Data warehousing and ETL (Extract, Transform, Load)
- Business reporting and data visualisation
- Data mining — classification and clustering
- Predictive modelling techniques
- Text analytics, text mining, and sentiment analysis
- Web analytics and social analytics
- Automated decision systems
- PowerBI dashboards and business analysis

## Tools and Technologies
